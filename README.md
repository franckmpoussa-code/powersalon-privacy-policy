# PowerSalon Privacy Policy

Site GitHub Pages prêt à publier.

## Fichiers
- `index.html` : page de politique de confidentialité
- `logo.png` : logo de l'application

## Publication sur GitHub Pages
1. Créez un repository public : `powersalon-privacy-policy`
2. Importez `index.html` et `logo.png`
3. Allez dans `Settings` → `Pages`
4. Source : `Deploy from a branch`
5. Branch : `main` / `/root`
6. Cliquez sur `Save`

URL finale attendue :
`https://VOTRE-NOM-GITHUB.github.io/powersalon-privacy-policy/`
